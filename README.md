# Developer Guide

Welcome to the laamly development team, where we're building the social media platform of the future™


To get start, have `node` and `yarn` installed. `npm` does work instead of `yarn` if necessary.
There are two parts to the project - the Vite frontend and the NodeJS backend, and both need to run for the site to work fully. Each has it's own `.env` file in the root of the directory, which you can get from the pinned messages in the Discord.


To get start, run `yarn install` in both directories, then `npm run dev` in both directories as well. The Vite frontend will tell you which localhost port the site was opened on, you can access it from there.
